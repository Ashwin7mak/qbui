Re-Arch UI Coding Style Guide

  1. Empty blocks are not allowed, except for catch blocks.

    DO THIS 
    if (a === b) {c = true }
try {a = b} catch(e) {};
    NOT THIS 
    if (a ===b_ {} else {d == false}

  2. Do not use the following keywords:

    • "with"

  3. Do not place the following keywords on a new line:

    • "else"
    • "catch"
    DO THIS 
    } else {
    NOT THIS 
    }
else {

  4. Lines may not contain a mixture of spaces and tab (\t) characacters.

    DO THIS 
       x = 1; // \t
   y = 2; // \s\s\s\s
    NOT THIS 
         z = 3; // \t\s

  5. Do not put any spaces after a unary prefix operator.

    DO THIS 
    var x = ++y;
    NOT THIS 
    var x = ++ y;

  6. There must not be a space before the opening paren in a function expression.

    DO THIS 
    var x = function() {};
    NOT THIS 
    var x = function () {};

  7. In an Array, do not put a space after the opening square bracket, or before the closing square bracket.

    DO THIS 
    var x = [1, 2];
var x = [1, [2, 3]];
    NOT THIS 
    var x = [ 1, 2];
var x = [1, 2 ];
var x = [1, [ 2, 3 ] ];

  8. In an expression, do not put a space after the opening paren, or before the closing paren.

    DO THIS 
    var x = (3 + 4) * 5;
var x = (3 * (4 + 5)) * 6;
    NOT THIS 
    var x = ( 3 + 4);
var x = (3 + 4 );
var x = (3 + ( 4 * 5) ) / 6;

  9. All constructors except for this must be capitalized.

    DO THIS 
    var x = new A();
var y = new this();
    NOT THIS 
    var x = new a();

  10. When a list is split over multiple lines, the comma separator must come at the end of a line.

    DO THIS 
    var x = {
    one: 1,
    two: 2,
    three: 3
};
    NOT THIS 
    var x = {
    one: 1
    ,two: 2
    ,three: 3
};

  11. Curly braces are required after the following statements:

    • "if"
    • "else"
    • "for"
    • "while"
    • "do"
    • "try"
    • "catch"
    DO THIS 
    if (condition) {
    someThing();
}
    NOT THIS 
    if (condition) someThing();

  12. A space is required after all binary operators.

    DO THIS 
    x + y;
    NOT THIS 
    x +y;

  13. The following keywords must be followed by a space:

    • "if"
    • "else"
    • "for"
    • "while"
    • "do"
    • "switch"
    • "case"
    • "return"
    • "try"
    • "catch"
    • "typeof"
    DO THIS 
    if (x) {
    x++;
}
    NOT THIS 
    if(x) {
    x++;
}

  14. Do not allow 'sticky' binary operators. The following operators must be preceeded by a space:

    DO THIS 
    x != y;
    NOT THIS 
    x!= y;

  15. A space is required before the opening brace of a block statement.

    DO THIS 
    if (condition) {
    foo();
}
    NOT THIS 
    if (condition){
    foo();
}

  16. In the conditional (ternary) operator, spaces are to be included as shown.

    DO THIS 
    var x = a ? b : c;
    NOT THIS 
    var x = a?b:c;

  17. Block statements must be indented with 4 space characters.

    DO THIS 
    if (a) {
    b = c; \\ line is indented with 4 spaces
}
    NOT THIS 
    if (a) {
      b = c; \\ line is indented with 6 spaces
}

  18. Strings must be quoted with single-quote (apostrophe).

    DO THIS 
    var s = 'A String Value';
    NOT THIS 
    var s = "A String Value";
